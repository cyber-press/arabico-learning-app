# Arabico - Arabic Learning App

This is a React Native + Expo app to learn Arabic. Includes flashcards, quizzes, tracing (SVG), audio playback, and daily streak tracking.

## Getting Started

1. Install dependencies:
   npm install

2. Start the project:
   npx expo start

3. Export for web:
   npx expo export:web

## GitHub Deployment

Push this repo to your GitHub and optionally set up GitHub Pages for the web version.
